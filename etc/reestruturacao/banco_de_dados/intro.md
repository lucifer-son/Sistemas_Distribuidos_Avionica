# Regra geral do Visualizador Web do Banco de Dados

- O sistema disponibilizará uma interface web simples e direta para consulta das tabelas do banco de dados PostgreSQL.
- O serviço rodará localmente em uma porta específica (ex: `localhost:8081`), servindo como um painel administrativo.
- Seu propósito principal é permitir que o usuário, desenvolvedores e avaliadores do projeto visualizem os dados salvos sem a necessidade de instalar e configurar clientes de banco de dados externos (DBeaver, pgAdmin, etc.).
- A interface apresentará os dados estruturados de telemetria, rotas e eventos do sistema que foram persistidos pelo backend.
