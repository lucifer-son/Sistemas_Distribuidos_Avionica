# CDU-02: Tela de simulação

## Fluxo principal

1. Ao acessar a tela de simulação atraves do `/simulacao`, o usuario verar o `Menu` superio.

2. em baixo do `menu` terar um campo de pesquisa com uma lista de `aeronave`, aonde pedirar para que o usuario informe o `codigo da aronave` esse codigo sera obitido atraves do sistema de `torre de controle`.

3. Após o usuario selecionar a `aeronave`, e confirmar a simulação para aquela `aeronave` atraves de um botão chamado `Selecionar`. O sistema redirecionar para a tela de seleção de rota `simulacao/rota`.
